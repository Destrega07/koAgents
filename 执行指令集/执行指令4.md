# 核心开发任务说明

## 1. 拦截“解析”动作并调用文件流接口
在合作者卡片的【编辑状态】下，用户会点击 `{上传对话文档}`（支持 txt 文件），随后点击 `{解析}` 按钮[cite: 3, 5]：
- **前置动作**：拦截 `{解析}` 点击事件，获取用户上传的文件流，将其通过现有的文件上传和 `/api/chat` API 异步流式发送给对应的 Coze Bot（Bot ID：`7613766901940011048`）[cite: 2, 5, 6]。
- **上下文拼接**：在发送给 Coze 的参数包中，自动将卡片中已有的 `name`（姓名）和 `position`（职位）作为上下文提示拼接进去[cite: 5]。
- **加载态提示**：在卡片内部展现“AI 正在进行全方位特征扫描与分析中…”的 Loading 动效[cite: 12]。

## 2. 扣子原始内容折叠区（调试沙盒）
- 仿照【需求线索】子模块，在【合作者档案】页面面板的最下方（或列表根容器下方），新建一个使用 `<details>` 和 `<summary>` 标签实现的组件：**“Coze端原始对话分析内容折叠区（调试）”**。
- 只要接收到 Coze 返回的文本流，必须无条件地将原始文本完整塞入该折叠区中，确保长效调试透明度。

## 3. 核心算法：高容错反向文本解析器（Markdown Parser）
你需要编写一个精准的正则表达式解析函数，捕获 Coze 返回的标准 Markdown。请根据以下规则抓取内容并对齐系统的 `storage.js` 数据库字段[cite: 5, 13]：
- **日期解析**：定位 `- **对话日期**：(.*?)\n`。若捕获到 8 位纯数字（如20260613），则将其赋值给系统的 `commUpdatedYmd` 字段；若无法解析到，则默认获取当前系统的 YYYYMMDD 日期填入[cite: 13]。
- **性格特征与置信度**：
  - 定位 `- **性格特征**：(.*?)\n`，提取后赋值给 `personality`[cite: 7, 13]。
  - 定位 `- **性格置信度**：(.*?)\n`，提取百分比数值（如"80%"），赋值给 `personalityConfidence`[cite: 7, 13]。
- **沟通风格与置信度**：
  - 定位 `- **沟通风格**：(.*?)\n`，提取后赋值给 `commStyle`[cite: 7, 13]。
  - 定位 `- **风格置信度**：(.*?)\n`，提取后赋值给 `commStyleConfidence`[cite: 7, 13]。
- **决策偏好与置信度**：
  - 定位 `- **决策偏好**：(.*?)\n`，提取后赋值给 `decisionPreference`[cite: 7, 13]。
  - 定位 `- **决策置信度**：(.*?)\n`，提取后赋值给 `decisionPreferenceConfidence`[cite: 7, 13]。
- **动态关系与建议**：
  - 定位 `- **关系变化提示**：(.*?)\n` $\rightarrow$ 赋值给 `relationChangeHint`[cite: 7, 13]。
  - 定位 `1. (.*?)\n`、`2. (.*?)\n`、`3. (.*?)\n` $\rightarrow$ 捕获为 3 个字符串元素构成的数组，赋值给 `commSuggestions`（对应系统字段说明中的“三条沟通建议”）[cite: 7, 13]。
  - 定位 `- **其他提醒**：(.*?)$` $\rightarrow$ 赋值给 `otherNotes`[cite: 7, 13]。
- **系统自增计算**：解析成功落库时，系统需自动将该合作者对象的 `analysisSamples`（对话分析累积样本数）自增 `+1`[cite: 13]。

## 4. 联动落库与视图状态回转
- 解析完成后，隐式调用 `updateCollaborator(id, parsedPatch)` 更新 IndexedDB 本地数据仓[cite: 7]。
- 自动将当前合作者卡片的视图状态从“编辑（edit）”回转切换为“展开（expanded）”[cite: 5]，使用户能够立即通过环形进度条和卡片面板一眼看到刷新后的所有“AI填写”数据，并弹出 `toast("AI 分析已成功回填并记录档案")`[cite: 5, 13]。

# 技术要求
1. 保持纯原生 JS 编写，不引入任何外部 Markdown 第三方解析库。
2. 确保正则表达式对可能出现的空格、换行、无序列表符号（`-` 或 `*`）具有高鲁棒性包容。
3. 编码完成后，请交付核心的解析器正则表达式代码块。
