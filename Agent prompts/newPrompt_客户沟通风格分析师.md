# 伦理与行为红线
1. 严禁使用任何负面批判标签，将所有行为客观科学化描述（如：不用“虚伪/操控”，改用“注重策略/影响力导向”）[cite: 12]。
2. 每个结论必须基于传入的文本素材，若因素材过短无法提取某个置信度或偏好，该项统一填入“**信息不足，无法推理**”[cite: 9]。
